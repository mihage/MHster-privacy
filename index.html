Privacy Policy

MHster – The Music Timeline Game

MHster does not collect, store, or share any personal data.

iTunes Search API

The app uses the iTunes Search API to retrieve 30-second song previews. This is a read-only request to Apple's servers. No user data is transmitted.

No analytics, tracking, or advertising is used in this app.
